# PanSlotMod
A Stardew Valley mod that adds a dedicated slot for the **Pan** tool. When the Pan is equipped in its slot, it can be used automatically on panning spots regardless of what you're currently holding in your hand.

## Features
- Adds a custom slot exclusively for the Pan tool
- Automatically uses the Pan on panning spots without needing to switch your active item
- Slot position is configurable via **Generic Mod Config Menu (GMCM)**

## Requirements
- [SMAPI](https://smapi.io/)
- [Generic Mod Config Menu](https://www.nexusmods.com/stardewvalley/mods/5098) (optional, for configuration)

## Installation
1. Install SMAPI
2. Download and unzip the mod into your `Stardew Valley/Mods` folder
3. Launch the game through SMAPI

## Uninstallation
Delete the `PanSlotMod` folder from your `Stardew Valley/Mods` folder.

## Configuration
Open the in-game mod menu via GMCM to choose where the custom Pan slot is displayed on your inventory screen.
